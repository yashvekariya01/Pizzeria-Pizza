# 🍕 Pizzeria-Pizza (Android App)

## About The App
Pizzeria-Pizza is an Android-based pizza ordering app designed for restaurants and small pizza shops.
This app supports dine-in and takeaway orders only (no delivery system).

Customers can browse the menu, customize pizzas, add items to cart, and place orders easily.
Admin can manage pizzas, toppings, and view orders from the admin panel.

---

## Features

### Customer Side
- View pizza menu
- Customize pizza (size, crust, toppings, cheese level)
- Add to cart
- Choose order type (Dine-in / Takeaway)
- Select pickup time
- Payment option (Cash / UPI)
- Order confirmation screen

### Admin Panel
- Admin login
- Add / edit / delete pizza
- Manage toppings
- Add categories
- View orders

---

## Not Included
- No delivery system
- No order tracking screen
- No offers or coupons
- No notifications
- No ratings and reviews
- No billing system
- No sales report
- No user management
