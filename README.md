# 🍕 Pizzeria-Pizza  
**A Smart Digital Solution for Modern Pizza Outlets** Pizzeria-Pizza is a robust Android application designed to bridge the gap between pizza shop owners and their customers. It provides a seamless digital interface for browsing menus, placing orders (Dine-in/Takeaway), and collecting customer insights, all while functioning efficiently without the need for constant internet connectivity.

---

## ✨ Key Features

- **🚀 Dual Ordering Modes:** - **Dine-in:** Order directly from the table.
  - **Takeaway:** Pre-order for quick pick-up.
- **📦 Offline Data Management:** Powered by **SQLite**, ensuring all orders and feedback are saved securely without internet.
- **👨‍💼 Admin Dashboard:** Digitally track incoming orders and manage pizza listings.
- **⭐ Feedback System:** Built-in rating and review section for customer engagement.
- **📞 Direct Support:** One-tap Call and Email support for instant communication.

---

## 🛠 Tech Stack

- **Language:** Java (Android SDK)
- **UI Design:** XML (Material Design Components)
- **Database:** SQLite (Local Persistent Storage)
- **Architecture:** Activity-based with Navigation Drawer
